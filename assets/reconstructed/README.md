# Reconstructed Assets (assets/reconstructed/)

This directory contains original recreations of assets from the original Delay Lama plugin and website.

The goal is to reproduce the original material as accurately as possible while creating the assets completely from scratch. Over time this directory may contain, among other things:

- SVG recreations of the original artwork and logos.
- Blender (`.blend`) source files for reconstructed models.
- Recreated textures, interface graphics, and other visual assets.
- Any additional assets produced as part of documenting or preserving the original plugin.
