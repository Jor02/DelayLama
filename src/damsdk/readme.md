# DamSDK

DamSDK is a lightweight, independent synthesis plugin interface i'm developing alongside the delay lama remake. 

By leveraging insights gained from reverse engineering legacy VST architectures, DamSDK provides a streamlined alternative to existing frameworks. It is designed for developers who need high compatibility without the overhead of proprietary SDKs.

### Project Status
Currently, DamSDK is being developed as an integrated component of the Delay Lama remake. As the specification matures and the API stabilizes, it will be migrated to a dedicated standalone repository for broader community use (if there's any interest for it that is).
