# Original Delay Lama Files

This directory contains the original Delay Lama related files, including the original documentation and the direct Ghidra exports.

## Contents

- **decomp/** - Contains unprocessed headers I've directly exported from Ghidra, I will be using this as a refernce for what classes I need to add. 
- **docs/** - Contains documentation and press releases
  - [manual.md](./docs/manual.md) - User manual for the Delay Lama plugin
  - [press_release.md](./docs/press_release.md) - Original press release
  - [picz/](./docs/picz/) - Image files used in the documentation

## Purpose

This folder preserves the original Delay Lama files for reference and historical purposes. Delay Lama is a popular vocal synth released in 2002 that offered a 'real-time' animated 3D interface.

## Usage

The files in this directory are provided as-is for reference. The original Delay Lama plugin can be used in compatible host applications that support the plugin standard.
